Getting started:

Here are the steps to get started with shapy.js: 1. Download the code from the github repository. 2. In the head of your html page, add the following two lines in order: The first is to include jquery and the second is to include shapy.js 3. In your javascript file, add this line at the top: const shapy = new Shapy() Now you can call all of shapy's methods using the new shapy object. Ex: const shapy = new Shapy() const element = document.querySelector(".elem") const elementRotate = shapy.rotateable(element) 4. Now just include your javascript file in your html file after the line including shapy.js, and you're all set! API

Functions

Name Parameters Return Description
animateSpriteWithClass class_: String,
name: String,
num: int,
interval: int,
stopElement: Object		 null Takes in a class of elements with class name given by class_, name is the path to your images + the image prefix name, num is the number of images in the folder, interval is the time in ms after which the image is changed, and stop element is the element which will be clicked to stop the animation. When this function is called, the elements with the given class name are animated by displaying the images in the given images folder sequentially on the given time interval. This process repeats until the stop element is pressed.
animateSpriteWithId id: String,
name: String,
num: int ,
interval: int,
stopElement: Object		 null Takes in an element with id given by id, name is the path to your images + the image prefix name, num is the number of images in the folder, interval is the time in ms after which the image is changed, and stop element is the element which will be clicked to stop the animation. When this function is called, the element with the passed in id, is animated by displaying the images in the given images folder sequentially on the given time interval. This process repeats until the stop element is pressed.
draggable Object null Calling this function on an object gives it drag and drop functionality. Only use this class on an object with absolute positioning and with a parent that does not have relative positioning.
draggableRelative Object null Calling this function on an object gives it drag and drop functionality. Only use this class on an object with absolute positioning and with a parent that has relative positioning.
positionElementCircle className: String,
centerX: float,
centerY: float,
radius: float		 null This function takes in a class name, and a specification for a circle with radius given by radius and center given by centerX and centerY. The elements belonging to className, will be positioned absolutely according to the circle specification that was passed in.
positionElementSquare className: String,
centerX: float,
centerY: float,
width:float,
height: float		 null This function takes in a class name, and a specification for a rectangle with width given by width, height given by height and center given by centerX and centerY. The elements belonging to className, will be positioned absolutely according to the rectangle specification that was passed in.
rotateable Object null An object of this class can be rotated by clicking on it and dragging the mouse to the desired angle of rotation.
rotateElements className: string,
stopElement: Object,
interval: int,
transition: String		 null This function takes in a class name an object designated as the stop object, a time interval and a css transition property. When this function is called, all elements with the given class name will rotate to the position of the next element on the specified interval using the specified transition property. Press the stopElement to stop the rotation.
resizeable Object null An object of this class is given four resize handles, one on each corner. Drag the handle to resize the object.
